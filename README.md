# Nutikas restorani reserveerimissüsteem - CGI proovitöö

## Kasutatud tehnoloogiad

* **Backend:** Java 21, Spring Boot 3, Gradle
* **Andmebaas:** PostgreSQL
* **Dokumentatsioon:** Swagger (OpenAPI)
* **Frontend:** Vue.js (lisandub hiljem)

## Päevik ja ajakulu

### Päev 1

* **Tegevused:** Tegevuskava loomine, Projekti algatamine, PostgreSQL seadistamine, Swaggeri integreerimine, Giti hoidla
  loomine.
* **Ajakulu:** ~ 45 minutit

### Päev 2

* **Tegevused:** Restorani plaani ja filtrite läbimõtlemine, DB tabelite täitmine andmetega, entity klasside loomine.
* **Ajakulu:** ~ 3 tundi

### Päev 3

* **Tegevused:** getTables funktsionaalsuse loomine, silmitsi seismine probleemiga, kus rakendus oli ühendatud vale
* andmebaasi instanstiga (oli väga õpetlik probleemilahendamine)
* **Ajakulu:** ~ 2 tundi
